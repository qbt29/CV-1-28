# Installation:
  - git clone https://github.com/qbt29/CV-1-28.git
  - cd CV-1-28
  - pip3 install requirements.txt
# Usage:
  ## Flags:
    -f <path_to_file> - to save image with circled angles into file, without this flag - display via graph
    -p <path_to_file> - file on which the angles must be found, without this flag - angles would be found on skimage.coins
  ##  Run:
    python3 main.py <flags>
